# Интерпретатор командной строки на C

Интерактивный командный интерпретатор (shell), реализованный на языке C. Выполняет разбор пользовательского ввода, запуск программ, поддержку конвейеров, фоновых процессов и перенаправления ввода-вывода.

## Возможности

### Разбор пользовательского ввода
- ✅ Чтение строк произвольной длины
- ✅ Разделение на слова по пробельным символам
- ✅ Поддержка двойных кавычек (текст в кавычках — одно слово)
- ✅ Обработка специальных символов: `>`, `<`, `>>`, `|`, `&`, `;`, `:`, `,`, `!`, `$`, `^`
- ✅ Корректная обработка EOF (Ctrl+D)
- ✅ Формирование связного списка слов

### Выполнение команд
- ✅ Запуск внешних программ (поиск в PATH)
- ✅ Встроенная команда `cd` (смена текущего каталога)
- ✅ Встроенная команда `exit` (выход из интерпретатора)
- ✅ Фоновый режим (`&` в конце команды)
- ✅ Уведомление о завершении фоновых процессов с кодом возврата
- ✅ Конвейер из двух команд (`|`)
- ✅ Перенаправление ввода (`<`)
- ✅ Перенаправление вывода (`>`)
- ✅ Перенаправление вывода с добавлением (`>>`)
